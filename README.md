# My-Tableau-Project
## Project Overview

This project was completed as part of a **Data Technician Bootcamp** and demonstrates the use of **Tableau** to analyse and visualise retail and sales data. The objective was to transform raw data into meaningful insights by building interactive dashboards and clear visualisations that support data-driven decision making.

## Key Skills Demonstrated

- **Interactive Dashboards**
  - Designed dynamic dashboards that allow users to explore retail and sales data in an intuitive and visual way.
  - Organised visual elements to present key metrics and trends clearly.

- **Filters and Slicers**
  - Implemented interactive filters to allow users to explore data by category, region, time period, or product.
  - Enabled deeper analysis by allowing users to focus on specific segments of the dataset.

- **Calculated Fields**
  - Created custom calculations to derive new metrics and enhance the analytical depth of the visualisations.

- **Data Visualisation**
  - **Bar Charts** – Compare sales performance across categories or products.
  - **Line Charts** – Visualise trends and changes in sales over time.
  - **Pie Charts** – Show proportional distribution of sales across categories.
  - **Maps** – Display geographical sales patterns and regional performance.

## What the Project Demonstrates

Using a retail and sales dataset, this project demonstrates how Tableau can be used to:

- Transform raw data into interactive visual dashboards  
- Identify patterns and trends in sales performance  
- Compare product categories and regional results  
- Enable users to explore data dynamically using filters  
- Communicate insights through clear and engaging visuals  

## Learning Outcomes

Through this project, I developed practical experience in:

- Designing interactive Tableau dashboards  
- Creating calculated fields to enhance analysis  
- Using multiple chart types to present data effectively  
- Applying filters and interactive elements to improve user exploration  
- Using **data storytelling** techniques to communicate insights clearly  

This project highlights the development of **data visualisation and storytelling skills**, enabling complex retail and sales data to be presented in a clear, accessible, and insightful way.
